COM3D2.GizmoFix.Plugin

※重要※
非公式プラグインです、規約にも則っていません。
使用においては「個人の責任の下」使用してください。
またこのプラグインは実験版です、動作確認などが不完全です。

標準機能のGizmoRender(回転とか移動をさせるためにマウスでつまむやつ)が何故かつまめなくてストレスフルなので、
それをCopilotくんに解決して貰おうとした他力本願プラグイン。

ほぼほぼCopilotくん(中身はClaude Sonnet 4.6)が実装したもの。

1.回転用の半円を全円(全円化した部分は色を暗く)
※実は半円にしてるのが見た目とあたり判定がズレててつまめない原因じゃないかと思ったので

2.マウスのあたり判定を広げて
※そもそもあたり判定が小さすぎるのが原因じゃないかと思ったので

3.IK操作用のキューブ？の見かけ上のサイズを一定にする
・メイドさんに近寄ってもキューブ？が回転用の円を隠さない
・カメラを引いた状態で背景オブジェクトをつかむのもやりやすくなるかも？
※jp.fumble.GizmoRenderFix.cfgのIKDragPoint Settingsセクション、Sizeでサイズを変えられます。

4.IK操作用とそれ以外でサイズ設定を別に
・IK以外が小さくなりすぎだったのを解消

5.回転円でホバー表示するように
※掴もうとして掴めないのはイライラの原因になると思ったので

6.ホバーの対象を実際に掴めるGizmoだけに

7.線を太く表示できるように

8.FOVを変更した際にもG正しく掴めるように

なお、他のGizmoの動作/描画を変更するプラグイン/スクリプトとは競合します。
※おそらくFOVを変更した際に正しく動作するようにするプラグインだと思われるので、いずれかを選択して導入してください。
